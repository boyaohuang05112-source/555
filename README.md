# Dondon Vauxhall - Quality Ingredients

A standalone landing page showcasing Dondon Vauxhall's premium ingredients and brand philosophy.

## Features

- **Three-page structure**: Introduction, Who We Are, Join Us
- **Interactive ingredients showcase**: 18 main ingredients with floating animations
- **Responsive design**: Optimized for desktop, tablet, and mobile devices
- **Smooth scrolling**: Scroll-snap navigation between pages

## Usage

Simply open `quality-ingredients.html` in your web browser or serve it using any static file server.

### Local Development

You can use Python's built-in server:
```bash
python3 -m http.server 8000
```

Or use Node.js http-server:
```bash
npx http-server -p 8000
```

Then open `http://localhost:8000/quality-ingredients.html` in your browser.
